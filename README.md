## wmonitord

**wmonitord** it's a simple tool for watching and logging your activity...
```
wmonitord 1.0.0
Copyright (C) 2024 wmonitord
Arguments:
  d - dubug mode;
  update period (ms);
  save period (ms);
  single object life time in memomy(h);
```
**Example:**
```
wmonitord.exe d 5000 30000000 48
```
Log file will be in desktop.
